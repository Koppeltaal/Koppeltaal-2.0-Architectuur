# Inleiding

Deze site beschrijft de architectuur van Koppeltaal GGZ versie 2.0. De site is opgesteld onder verantwoording van VZVZ. De scope is een beschrijving van de architectuur op 'Enterprise Architectuur' niveau, waarbij het TOGAF-raamwerk, en de Architectuur Development Methode \(ADM\) zoveel mogelijk gebruikt worden als taal voor vastlegging van de architectuur.
